# Task Manager App (React)

A modern and interactive task management application built with React. This app demonstrates clean UI design, state management, and user-focused features.

---

## Features

- Add, edit, and delete tasks
- Mark tasks as completed with confetti animation 🎉
- Filter tasks (All, Pending, Completed - Recent)
- Local storage persistence (data saved after reload)
- Smooth UI animations and hover interactions
- Clean and minimal spring-themed UI

---

## Concepts Used

- React Hooks (useState, useEffect)
- Conditional rendering
- Component-based architecture
- LocalStorage integration
- UX/UI best practices

---

## Tech Stack

- React (Vite)
- JavaScript
- CSS (Custom Styling)
